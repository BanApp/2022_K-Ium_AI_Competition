# 📘 기능 설명

## 1) 모델학습_소스코드.ipynb 를 통해서 학습 데이터(csv)를 호출하고 데이터를 전처리후 모델을 학습한다.
## 2) 데이터_전처리_및_모델_호출_소스코드.ipynb 를 통해서 평가 데이터(csv)를 호출하고 데이터를 전처리후 모델을 사용한다.
